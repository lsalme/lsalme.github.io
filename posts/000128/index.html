Android 更改系统时区为 UTC

Jul 22, 2018 07:03 · 112 words · 1 minute read code android

Android 系统设置里不能把时区更改为 UTC，只能通过模拟终端或 ADB 解决。

两种方法都试了，还不清楚是哪种产生的效果。重启之后时区改变。

方法一

adb shell
su
setprop persist.sys.timezone UTC
exit
exit

方法二

adb shell
su
settings put global time_zone UTC
exit
exit
