帮同学装了 Ubuntu，之前装过很多次，但是似乎没有怎么注意一些问题。记录一下。

禁用 ipv6

由于校园网分配了 ipv6 地址，连接 ustc 的源的时候，会默认使用 ipv6 ，然而很慢，且经常不通。故选择禁用 ipv6 。

sudo vim /etc/default/grub

修改为：

GRUB_CMDLINE_LINUX_DEFAULT="ipv6.disable=1 quiet"

更新 grub ：

sudo update-grub

重启。

安装 nvidia 显卡驱动

由于无法调节屏幕亮度，估计是显卡驱动问题。

可先查看需要安装什么驱动：

sudo ubuntu-drivers list

安装：

sudo ubuntu-drivers autoinstall

重启即可。

安装 flash

sudo apt install flashplugin-installer -y

安装 qt

sudo apt install qt5-default qtcreator qtdeclarative5-dev -y
